# Ejercicio Práctico

Una empresa de seguros ofrece a sus **clientes** diferentes **servicios** para cubrir de cualquier accidente a sus **propiedades**.

* De cada **cliente** se solicita: `nombre`, `apellido`, `dni`, `fecha_nac` y `sueldo`.
* Cada **servicio** ofrecido cuenta con los siguientes datos: `código`, `tipo`, `detalle_cobertura`, `precio`.
* De cada **propiedad** se solicita conocer: `dirección`, `metros_cuadrados`, `zona`, `valor_mercado`.

A partir de este planteo:

* **Confeccionar el DER** (Diagrama Entidad Relación) de la situación planteada.
* **Crear una nueva base de datos** llamada `emp_seguros`.
* **Plasmar lo confeccionado en el DER** en la Base de datos creada.

Y ahora... ¿Estas entidades están aisladas o se relacionan entre sí?
